Multi-Agent Agricultural Network Module

This is a senior capstone project. The project delivered an modularized network module that were contracted by the Agricultural Robotic Department of the University of California, Davis. The goal of the project is to design and implement a network module that enables asynchronous bidirectional communications between client's stationed server and multiple edge devices scattered on the field in a mesh topology. The network module is fully encapsulated with an internal Python server to maintain the communication logics and custom message scheme, therefore we also provide the corresponding API for easy user interface.

The module was field tested with 14 edge nodes in the vineyard with the consideration of forage obstruction. In an unobstruted setting, a 20 byte message is able to reach 400 ft with closed to 0% package loss, and up to 700ft with one edge node that provides message hopping. It can reach 30-50ft through multiple rows of grape vine.

The project is done in a group of four with multiple comonents developed in parallel. While other members worked on the root and edge module of the BLE module and custom PCB, My responsibility was the network server used for caching data periodically updated from the edge nodes and other internal network logic such as disconnection failsafe.

I learned a lot about socket programming such how to apply observer pattern with sckets to handle events asynchronously with threads. I also get to practice agile developments with weekly meetings with the team and the biweekly meetings with our client.

More detailed documentation can be found in the github page.
